## Key finding (reproducible: `gatot_1v1_isolation.py`, `gatot_ladder_analysis.py`)

The Gatot 1v1 single-target rows (exact turns) pin every exponent independently:
- **Attack** `turns*A ~ 301` const  -> damage ~ A^1
- **Defense** `turns/D ~ 21.0` const -> damage ~ 1/D
- **Health** `turns*A/D` linear in H -> HP ~ H^1
- **Lethality** (beast ladder) -> damage ~ L^1

**Combined:** `turns = 12.54 * D_def*H_def / (A_att*L_att)` fits all 15 captured-H
T1 rows to <=0.4 turns, and the T3 rows are internally consistent with the same law.

=> **Per-unit law: damage/turn ~ A_att*L_att / D_def ; HP pool ~ H_def**
   i.e. the **A*L/(D*H)** monomial, exponents (A,L,D,H)=(+1,+1,-1,-1).

This is the form Stage 3 REJECTED — because Stage 3 assumed ADDITIVE base-tier
stats (n,n+3,n,n+5). Real base stats scale MULTIPLICATIVELY (~x1.20/tier, from the
T3 rows), which makes A*L/(D*H) tier-invariant in mirrors (matches NanoMart flat
264/266) and explains the sublinear beast tier ladder. `HP = D+H` is refuted; the
structure is the product D*H (each exponent exactly 1), not the sum.
