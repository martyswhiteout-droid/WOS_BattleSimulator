# NanoMart vs MiniMart Deterministic Experiment Ledger

This is the single flat ledger for every deterministic NanoMart versus MiniMart battle-report fixture in `wos_sim/data/experiments` as of 2026-07-11. It contains 70 JSON report rows plus the unique 10v10 report that exists only as a structured Markdown capture. Alternative Markdown captures for three 1v1 reports are linked from their JSON row rather than counted twice.

The table preserves report inputs and outcomes. `Panel` gives the complete visible three-class A/D/L/H percentage panel; `NC` means the report did not display that field. Martin confirmed chief-panel Lethality and Health bonuses are 0% in this corpus, so the deployed-input column uses inherent tier L/H where those rows are absent. Hero and troop skills remain separate battle-time inputs.

Unless a filename says otherwise, Vulcanus skills are Level 1 and Seo-yoon Skill 1 is Level 3. `P` = power loss, `Loss` = losses, `Inj` = injured, `Light` = lightly injured, `Surv` = survivors, `K` = reported kills, and `T` = skill triggers.
